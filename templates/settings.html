{# ── MCP / AI agent ── #}

Run mcp_server.py locally — Claude Desktop will launch it automatically. It calls this server's API using your key.

1. Install dependencies
pip install mcp requests
2. Download the MCP server script
{{ t['settings_api_download_mcp'] }}
3. Claude Desktop config

Linux: ~/.config/Claude/claude_desktop_config.json
Mac: ~/Library/Application Support/Claude/claude_desktop_config.json 

{
  "mcpServers": {
    "plantlog": {
      "command": "python3",
      "args": ["/path/to/Plantlog/scripts/mcp_server.py"],
      "env": {
        "PLANTLOG_URL": "{{ request.host_url.rstrip('/') }}",
        "PLANTLOG_API_KEY": "<your-key>"
      }
    }
  }
}
{# ── Printer client ── #}

label_client.py is written for the YHK-835E Bluetooth thermal printer. If you use a different model, the Bluetooth handshake and ESC/POS commands in the script may need to be adjusted.

Run label_client.py on the machine physically connected to your printer. It polls this server for print jobs and sends them to the printer automatically.

1. Install dependencies
pip install requests Pillow
2. Download
{{ t['settings_api_download_py'] }}
{{ t['settings_printer_mac_hint'] }}
3. Run it

On first run it will ask for your server URL, API key, and printer MAC, then save a config file.

python3 label_client.py

Or set env vars directly:

PLANTLOG_URL={{ request.host_url.rstrip('/') }} \
PLANTLOG_API_KEY=<your-key> \
PRINTER_MAC=<your-printer-mac> \
python3 label_client.py
Config file location

Linux / Mac: ~/.config/plantlog/label_client.json
Windows: %APPDATA%\plantlog\label_client.json 

{
  "url": "{{ request.host_url.rstrip('/') }}",
  "api_key": "<your-key>",
  "printer_mac": "<your-printer-mac>",
  "printer_port": 2,
  "poll_interval": 5
}
Run as a background service (Linux systemd)
[Unit]
Description=Plantlog label client

[Service]
ExecStart=python3 /path/to/Plantlog/scripts/label_client.py
Environment=PLANTLOG_URL={{ request.host_url.rstrip('/') }}
Environment=PLANTLOG_API_KEY=<your-key>
Environment=PRINTER_MAC=<your-printer-mac>
Restart=always
RestartSec=10

[Install]
WantedBy=default.target

Save as ~/.config/systemd/user/plantlog-printer.service then: systemctl --user enable --now plantlog-printer
